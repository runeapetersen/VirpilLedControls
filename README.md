# A Virpil LED Control Script for SPAD.neXt

Since SPAD.neXt lacks the ability to control the LED lights on some Virpil devices this script is intended to fill the gap.

Highlights:
- Set a solid color on buttons via the SPAD.neXt application without depending on the Virpil software
- Create simple automated color cycles with a user-defined interval between color changes
- Works like any other script, meaning it can be triggered from any SPAD.neXt state change or rule you care to define
- instrumented using a simple JSON structure which is passed to the script
